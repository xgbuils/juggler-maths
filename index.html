\( \def \Nat {\mathbb{N}} \)

Definició 1:

Donat $n \in \Nat$ definim el conjunt de patrons vàlids $P_n$ de la següent manera: $$P_n = \{ x \in \Nat^n : \forall{i, j \in \{0..n-1\} ( \exists{q \in \Nat, r \in \{0..n-1\}^n (x_i + i = nq + r_i} \land ( r_i = r_j \implies i = j) ) } \} $$

Definició 2:

Donat $n \in \Nat$ definim la funció normalitzadora de patrons de periode n $\phi_n : \Nat^n \to \{0..n-1\}^n $ de la següent manera: $$\phi_n(x) = r : \forall{i \in \{0..n-1\} (\exists { q \in \Nat (x_i + i = qn + r_i )} ) } $$

Definició 3:

Donat $n \in \Nat$ definim $S_n$ com el conjunt de permutacions $f: \{0..n-1\} \to \{0..n-1\}.$

Definició 4:

Donat $n \in \Nat$ definim el conjunt de tuples de període $n$ sense repeticions $S'_n$ de la següent manera: $$S'_n = \{x \in \Nat^n : \forall i, j \in \{0..n-1\} (x_i = x_j \implies i = j) \}$$
