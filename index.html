Definició 1:

Donat $n \in \Nat$ definim el conjunt de patrons vàlids $P_n$ de la següent manera: $$P_n = \{ x \in \Nat^n : \Forall{ i, j \in \{0..n-1\} }{ \Exists{ q \in \Nat, r \in \{0..n-1\}^n }{ x_i + i = nq + r_i \land ( r_i = r_j \then i = j } } \} $$

Definició 2:

Donat $n \in \Nat$ definim la funció normalitzadora de patrons de periode n $\phi_n : \Nat^n \to \{0..n-1\}^n $ de la següent manera: $$\phi_n(x) = r : \Forall{ i \in \{0..n-1\} }{ \Exists { q \in \Nat }{ x_i + i = qn + r_i } } $$

Definició 3:

Donat $n \in \Nat$ definim $S_n$ com el conjunt de permutacions $f: \{0..n-1\} \to \{0..n-1\}.$

Definició 4:

Donat $n \in \Nat$ definim el conjunt de tuples de període $n$ sense repeticions $S'_n$ de la següent manera: $$S'_n = \{ x \in \Nat^n : \Forall { i, j \in \{0..n-1\} }{ x_i = x_j \then i = j } \}$$

Lema:

Sigui $\phi_n(P_n)$ la imatge de $P_n$ sobre $\phi_n$. Existeix $f: \phi_n(P_n) \to {S'}_n$ tal que $f$ es bijectiva.

Demostració 1:

Tenim que: $$\phi_n(P_n) = \{y \in \{0..n-1\}^n : x \in P_n \land \phi_n(x) = y \}$$

Per la definició 1 i 2: $$\phi_n(P_n) = \{ y \in \{0..n-1\}^n : x \in \Nat^n \land \Forall{ i, j \in \{0..n-1\} }{ \Exists{ q \in \Nat, r \in \{0..n-1\}^n }{ x_i + i = nq + r_i \land ( r_i = r_j \then i = j } } \land \Forall{ k \in \{0..n-1\} }{ \Exists { q \in \Nat }{ x_k + k = qn + y_k } } \}$$

Si $n = 0$, llavors $\{0..n-1\} = \emptyset$ i de manera trivial s'arriba a $\phi_n(P_n) = {S'}_n$

En cas contrari, podem aplicar les regles LPO 1.1 i LPO 2.1 transformant la expressió a: $$\phi_n(P_n) = \{ y \in \{0..n-1\}^n : \Forall{ i, j, k \in \{0..n-1\} }{ x \in \Nat^n \land \Exists{ q \in \Nat, r \in \{0..n-1\}^n }{ x_i + i = nq + r_i \land ( r_i = r_j \then i = j } \land \Exists { q \in \Nat }{ x_k + k = qn + y_k } } \}$$
